# Day 4 – Real-Time SOC Security Dashboard

In this stage, I built an advanced security dashboard to visualize system activity in real time.

The dashboard reads security logs continuously and displays multiple visualizations to help identify suspicious behavior.

## Dashboard Visualizations

Bar Chart
Shows failed login attempts per IP address to identify potential attackers.

Line Graph
Displays failed login activity over time to detect attack spikes.

Pie Chart
Shows the distribution of security events such as login_success, login_failed, and file_access.

## Technologies Used

Python  
Matplotlib  
CSV Log Analysis  

## Purpose

Security dashboards help SOC analysts quickly understand system activity and detect threats visually.

This stage simulates how monitoring dashboards are used in real security operations centers.
