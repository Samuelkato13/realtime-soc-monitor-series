## Real-Time SOC Monitoring System (Python)

This repository documents my learning journey building a **Real-Time Security Operations Center (SOC) monitoring system** using Python.

The project was built step-by-step over 5 days to simulate how security teams monitor systems and detect threats.

## What This System Does

✔ Generates realistic security logs  
✔ Monitors logs in real time  
✔ Detects suspicious login activity  
✔ Generates alerts for brute force attacks  
✔ Visualizes attack activity with dashboards  

## Project Progression

Day 1 – Log generation  
Day 2 – Real-time monitoring  
Day 3 – Threat detection  
Day 4 – Visualization dashboard  
Day 5 – Full SOC system demo
